# AndroidIcon

## 说明
将一个图标文件转换成Android的图标文件, 其中包含下面几种格式，并自动放在响应目录

	drawable-ldpi	36x36
	drawable-mdpi	48x48
	drawable-hdpi 	72x72
	drawable-xhdpi 	96x96
	drawable-xxhdpi	144x144
